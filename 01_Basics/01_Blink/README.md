# 01 - Blink (Hello World)

## Περιγραφή
Το κλασικό πρώτο πρόγραμμα Arduino. Αναβοσβήνει το ενσωματωμένο LED της πλακέτας κάθε 500ms.

## Πού χρησιμοποιείται σε υλοποίηση IoT;
*   **Heartbeat Συστημάτων:** Ένα LED που αναβοσβήνει σε IoT Gateways δείχνει ότι η συσκευή λειτουργεί και δεν έχει "κολλήσει".
*   **Ένδειξη Σύνδεσης:** Σε έξυπνες πρίζες, το γρήγορο αναβόσβημα συχνά δηλώνει λειτουργία σύζευξης (pairing mode) με το δίκτυο.
*   **Status Indicators:** Σε αισθητήρες χωρίς οθόνη, το LED επιβεβαιώνει τη λήψη μέτρησης.

## Μαθησιακοί Στόχοι
- Εκμάθηση της δομής ενός Arduino sketch (`setup()` και `loop()`)
- Κατανόηση των βασικών εντολών: `pinMode()`, `digitalWrite()`, `delay()`
- Επαλήθευση σωστής εγκατάστασης IDE, board και port

## Σχεδιάγραμμα Κυκλώματος

```
┌─────────────────┐
│   Arduino Uno   │
│                 │
│  D13 (Built-in) ├──► LED εσωτερικό της πλακέτας
│                 │
│      GND        │
└─────────────────┘
```

## Υλικά που Χρειάζονται
- 1× Arduino Uno (ή συμβατή πλακέτα)
- 1× Καλώδιο USB

**Σημείωση:** Δεν απαιτείται εξωτερική καλωδίωση!

## Οδηγίες Σύνδεσης

1. Συνδέστε το Arduino στον υπολογιστή με USB
2. Δεν χρειάζεται breadboard ή εξωτερικά εξαρτήματα
3. Το `LED_BUILTIN` (pin 13) έχει ενσωματωμένο LED και αντίσταση

## Πώς να το Τρέξετε

1. Ανοίξτε το `blink.ino` στο Arduino IDE
2. Επιλέξτε **Tools → Board → Arduino Uno**
3. Επιλέξτε **Tools → Port → COM#** (το σωστό port σας)
4. Πατήστε το κουμπί **Upload** (→)
5. Παρατηρήστε το LED στο pin 13 να αναβοσβήνει

## Επεξήγηση Κώδικα

```cpp
void setup() {
  pinMode(LED_BUILTIN, OUTPUT);  // Ρύθμιση pin 13 ως έξοδος
}

void loop() {
  digitalWrite(LED_BUILTIN, HIGH);  // Άναμμα LED
  delay(500);                       // Αναμονή 500ms
  digitalWrite(LED_BUILTIN, LOW);   // Σβήσιμο LED
  delay(500);                       // Αναμονή 500ms
}
```

- **setup()**: Εκτελείται μία φορά στην εκκίνηση
- **loop()**: Επαναλαμβάνεται συνεχώς
- **LED_BUILTIN**: Σταθερά που αντιστοιχεί στο pin 13
- **delay(ms)**: Παύση για ms χιλιοστά του δευτερολέπτου

## Πειραματισμός

Δοκιμάστε να αλλάξετε:
- Τις τιμές `delay(500)` σε `100` ή `1000` για διαφορετική ταχύτητα
- Διαφορετικές καθυστερήσεις για ON και OFF: `delay(200)` και `delay(800)`
- Προσθέστε πολλαπλά blinks σε ένα μοτίβο (π.χ. SOS: ···−−−···)
