# 09 Basics Capstone: Smart Light

Αυτό το έργο συνδυάζει όλες τις γνώσεις που αποκτήσατε μέχρι τώρα σε μία ολοκληρωμένη εφαρμογή.

## Πού χρησιμοποιείται σε υλοποίηση IoT;
*   **Smart Home Scenes:** Ένας έξυπνος διακόπτης τοίχου μπορεί να εναλλάσσει "σκηνές" φωτισμού (π.χ. Relax, Reading, Party) με διαδοχικά πατήματα.
*   **Wearables:** Σε fitness trackers, το πάτημα ενός κουμπιού αλλάζει την οθόνη από ώρα -> βήματα -> καρδιακούς παλμούς.
*   **Πλυντήρια IoT:** Η επιλογή προγράμματος πλύσης γίνεται συχνά κυκλικά με ένα κουμπί, αλλάζοντας την εσωτερική κατάσταση της συσκευής.

## Τι θα μάθετε
*   Πώς να συνδυάζετε πολλαπλές λειτουργίες (Input, Output, Timing, Serial) σε ένα πρόγραμμα.
*   Πώς να διαχειρίζεστε **Καταστάσεις (States)** για να ελέγχετε τη συμπεριφορά του συστήματος.
*   Πώς να γράφετε κώδικα που είναι **Non-blocking** (δεν κολλάει), ώστε το κουμπί να λειτουργεί άμεσα ακόμα και όταν το LED αναβοσβήνει.

## Κύκλωμα
*   **LED:** Εξωτερικό LED στο **Pin 9** (με αντίσταση 220Ω), όπως στα μαθήματα 07 & 08.
*   **Κουμπί:** Συνδεδεμένο στο **Pin 2** και στο GND (χρήση `INPUT_PULLUP`).

## Σχεδιάγραμμα Κυκλώματος

```
           ┌─────────────────┐
           │   Arduino Uno   │
           │                 │
  ┌────────┤ D2 (Input)      │
  │        │                 │
  │  ┌─────┤ D9 (Output)     │
  │  │     │                 │
  │  │ ┌───┤ GND             │
  │  │ │   └─────────────────┘
  │  │ │
  │  │ └──────┐
  │  │        │
  │  │       ─┴─  LED
  │  │       ▲ ▲
  │  │        │
  │  └────────ww────┐
  │         220Ω    │
  │                 │
  └─────────────────┘  Κουμπί
      (Πατήστε για GND)
```

## Λειτουργία
1.  Το σύστημα ξεκινάει σε κατάσταση **OFF**.
2.  Πατώντας το κουμπί, περνάει σε **SLOW_BLINK** (αργό αναβόσβημα).
3.  Πατώντας ξανά, περνάει σε **FAST_BLINK** (γρήγορο αναβόσβημα).
4.  Πατώντας ξανά, επιστρέφει στο **OFF**.
5.  Μπορείτε επίσης να στείλετε εντολές (`0`, `1`, `2`) μέσω του Serial Monitor.
