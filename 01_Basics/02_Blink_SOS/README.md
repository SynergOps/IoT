# 01_Blink_SOS - Σήμα Κινδύνου SOS

## Περιγραφή
Προσομοιώνει το διεθνές σήμα κινδύνου **SOS** σε κώδικα Morse χρησιμοποιώντας το ενσωματωμένο LED του Arduino. 

**SOS στον κώδικα Morse:**
```
S = ... (3 κοντά)
O = --- (3 μακριά)
S = ... (3 κοντά)
```

## Πού χρησιμοποιείται σε υλοποίηση IoT;
*   **Διάγνωση Σφαλμάτων (Error Codes):** IoT συσκευές χωρίς οθόνη αναβοσβήνουν με συγκεκριμένα μοτίβα για να δηλώσουν προβλήματα (π.χ. 3 γρήγορα = αποτυχία σύνδεσης Wi-Fi).
*   **Firmware Update:** Κατά τη διάρκεια αναβάθμισης λογισμικού (OTA), το LED αναβοσβήνει με συγκεκριμένο ρυθμό για να προειδοποιήσει τον χρήστη να μην αποσυνδέσει τη συσκευή.
*   **Οπτική Ειδοποίηση:** Έξυπνες κάμερες αναβοσβήνουν όταν καταγράφουν ή όταν εντοπίσουν κίνηση.

4. Παρατηρήστε το LED να εκπέμπει: **··· ─── ···** (SOS)

## Επεξήγηση Κώδικα

### Συναρτήσεις (Functions)
Χρησιμοποιούμε δικές μας εντολές για να κάνουμε τον κώδικα πιο καθαρό:
- `dot()`: Ανάβει το LED για σύντομο χρονικό διάστημα (Τελεία)
- `dash()`: Ανάβει το LED για μεγαλύτερο χρονικό διάστημα (Παύλα)

### Σταθερές Χρονισμού
```cpp
int DOT_DURATION = 200;        // Τελεία: 200ms
int DASH_DURATION = 600;       // Παύλα: 600ms (3× τελεία)
int SYMBOL_SPACE = 200;        // Διάστημα μεταξύ . και -
int LETTER_SPACE = 600;        // Διάστημα μεταξύ S-O-S
int WORD_SPACE = 1400;         // Παύση πριν επαναλάβει
```

### Συναρτήσεις για Επαναχρησιμοποίηση

```cpp
// Αντί να γράφουμε πολλές φορές τα ίδια:
void dot() {
  digitalWrite(LED, HIGH);
  delay(DOT_DURATION);
  digitalWrite(LED, LOW);
  delay(SYMBOL_SPACE);
}

void dash() {
  digitalWrite(LED, HIGH);
  delay(DASH_DURATION);
  digitalWrite(LED, LOW);
  delay(SYMBOL_SPACE);
}
```

✅ **Πλεονεκτήματα:**
- Ο κώδικας είναι πιο κατανοητός
- Εύκολη αλλαγή χρονισμών (1 θέση)
- Επαναχρησιμοποίηση για άλλα μηνύματα
