# 01_Blink_SOS - Σήμα Κινδύνου SOS

## Περιγραφή
Προσομοιώνει το διεθνές σήμα κινδύνου **SOS** σε κώδικα Morse χρησιμοποιώντας το ενσωματωμένο LED του Arduino. 
Αφού μάθαμε να αναβοσβήνουμε το LED, τώρα θα κάνουμε κάτι πιο σύνθετο. Θα προγραμματίσουμε το Arduino να αναβοσβήνει το ενσωματωμένο LED ακολουθώντας το μοτίβο του σήματος κινδύνου SOS στον κώδικα Morse.

Το SOS στον κώδικα Morse είναι: `... --- ...`
- Τρεις γρήγορες λάμψεις (S)
- Τρεις αργές λάμψεις (O)
- Τρεις γρήγορες λάμψεις (S)

**SOS στον κώδικα Morse:**
```
S = ... (3 κοντά)
O = --- (3 μακριά)
S = ... (3 κοντά)
```
## Σχεδιασμός Προγράμματος

Για να το πετύχουμε, μπορούμε να ορίσουμε δύο μονάδες χρόνου:
1.  `dot`: Η διάρκεια για μια τελεία (π.χ., 250 ms).
2.  `dash`: Η διάρκεια για μια παύλα (π.χ., 750 ms).

Θα χρειαστούμε επίσης παύσεις:
- Μικρή παύση μεταξύ των σημάτων του ίδιου γράμματος.
- Μεγαλύτερη παύση μεταξύ των γραμμάτων (S, O, S).
- Ακόμα μεγαλύτερη παύση πριν την επανάληψη του μηνύματος SOS.

## Πού χρησιμοποιείται σε υλοποίηση IoT;
*   **Διάγνωση Σφαλμάτων (Error Codes):** IoT συσκευές χωρίς οθόνη αναβοσβήνουν με συγκεκριμένα μοτίβα για να δηλώσουν προβλήματα (π.χ. 3 γρήγορα = αποτυχία σύνδεσης Wi-Fi).
*   **Firmware Update:** Κατά τη διάρκεια αναβάθμισης λογισμικού (OTA), το LED αναβοσβήνει με συγκεκριμένο ρυθμό για να προειδοποιήσει τον χρήστη να μην αποσυνδέσει τη συσκευή.
*   **Οπτική Ειδοποίηση:** Έξυπνες κάμερες αναβοσβήνουν όταν καταγράφουν ή όταν εντοπίσουν κίνηση.

## Επεξήγηση Κώδικα

### Συναρτήσεις (Functions)
Χρησιμοποιούμε δικές μας εντολές για να κάνουμε τον κώδικα πιο καθαρό:
- `dot()`: Ανάβει το LED για σύντομο χρονικό διάστημα (Τελεία)
- `dash()`: Ανάβει το LED για μεγαλύτερο χρονικό διάστημα (Παύλα)

### Σταθερές Χρονισμού
```cpp
int DOT_DURATION = 200;        // Τελεία: 200ms
int DASH_DURATION = 600;       // Παύλα: 600ms (3× τελεία)
int SYMBOL_SPACE = 200;        // Διάστημα μεταξύ . και -
int LETTER_SPACE = 600;        // Διάστημα μεταξύ S-O-S
int WORD_SPACE = 1400;         // Παύση πριν επαναλάβει
```

### Συναρτήσεις για Επαναχρησιμοποίηση

```cpp
// Αντί να γράφουμε πολλές φορές τα ίδια:
void dot() {
  digitalWrite(LED, HIGH);
  delay(DOT_DURATION);
  digitalWrite(LED, LOW);
  delay(SYMBOL_SPACE);
}

void dash() {
  digitalWrite(LED, HIGH);
  delay(DASH_DURATION);
  digitalWrite(LED, LOW);
  delay(SYMBOL_SPACE);
}
```

- **Custom Functions (Προσαρμοσμένες Συναρτήσεις)**: Αντί να γράφουμε τον ίδιο κώδικα ξανά και ξανά μέσα στο `loop()`, δημιουργήσαμε δύο νέες συναρτήσεις: `dot()` και `dash()`. Αυτό κάνει τον κώδικά μας:
    - **Πιο ευανάγνωστο**: Το `loop()` μοιάζει με την περιγραφή του προβλήματος (S, O, S).
    - **Πιο οργανωμένο**: Η λογική για την τελεία και την παύλα είναι τακτοποιημένη στη δική της συνάρτηση.
    - **Πιο εύκολο στη συντήρηση**: Αν θέλουμε να αλλάξουμε τη διάρκεια της τελείας, το αλλάζουμε μόνο σε ένα σημείο.

✅ **Πλεονεκτήματα:**
- Ο κώδικας είναι πιο κατανοητός
- Εύκολη αλλαγή χρονισμών (1 θέση)
- Επαναχρησιμοποίηση για άλλα μηνύματα

## Γλώσσα προγραμματισμού Arduino
Το Arduino χρησιμοποιεί μια γλώσσα προγραμματισμού που βασίζεται κυρίως στις C και C++.

Για την ακρίβεια:
- Η Βασική Γλώσσα: Η γλώσσα προγραμματισμού του Arduino είναι μια απλοποιημένη έκδοση της C++, με ορισμένες τροποποιήσεις για να είναι πιο εύκολη στη χρήση από αρχάριους.
- Το Framework (Πλαίσιο Εργασίας): Το Arduino IDE (Integrated Development Environment) παρέχει ένα σύνολο προ-εγκατεστημένων συναρτήσεων και βιβλιοθηκών (γνωστό ως το "Arduino Wiring Framework") που κρύβουν την πολυπλοκότητα των C/C++ από τον χρήστη. Αυτό σας επιτρέπει να γράφετε κώδικα με απλές εντολές όπως digitalWrite() ή Serial.begin(), αντί να χρειάζεται να χειρίζεστε απευθείας τους καταχωρητές (registers) του μικροελεγκτή.
- Άλλες γλώσσες: Αν και η C/C++ είναι η προεπιλογή, είναι δυνατόν να προγραμματίσετε το Arduino και με άλλες γλώσσες (π.χ. MicroPython, Assembly, κ.ά.), αλλά αυτό απαιτεί πιο εξειδικευμένα εργαλεία και είναι σπανιότερο.

Συνοψίζοντας, γράφετε κώδικα σε C++, αλλά μέσα σε ένα φιλικό περιβάλλον που λέγεται "Arduino language".
