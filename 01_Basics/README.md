## Γενικές Οδηγίες Υλικών και Καλωδίωσης

### Πολικότητα LED
```
     LED (Light Emitting Diode)
     
  Μακρύ πόδι (+)  │  Κοντό πόδι (-)
    Anode         │    Cathode
       │          │       │
       └──────┬───┴───────┘
              │
       Επίπεδη πλευρά (-)
```
- **Anode (+)**: Μακρύ πόδι → συνδέεται στο pin (μέσω αντίστασης)
- **Cathode (-)**: Κοντό πόδι ή επίπεδη πλευρά → συνδέεται στο GND

### Τιμές Αντιστάσεων (Χρωματικός Κώδικας)
| Τιμή | Χρώματα | Χρήση |
|------|---------|-------|
| 220Ω | Κόκκινο-Κόκκινο-Καφέ | LED (κόκκινο/πράσινο/κίτρινο) |
| 330Ω | Πορτοκαλί-Πορτοκαλί-Καφέ | LED (μπλε/λευκό) |
| 10kΩ | Καφέ-Μαύρο-Πορτοκαλί | Pull-down για κουμπιά |

### Συνήθη Λάθη
- ❌ LED αντίστροφα (δεν ανάβει ή καίγεται)
- ❌ Ξεχασμένη αντίσταση σε σειρά με LED (καίγεται το LED)
- ❌ Όχι κοινή γείωση (GND) μεταξύ συσκευών
- ❌ PWM pins: μόνο pins με ~ (D3, D5, D6, D9, D10, D11 στο Uno)
- ❌ Analog pins: A0-A5 για `analogRead()`, όχι για `analogWrite()`

### Breadboard Tips
```
Power Rails:          Internal Connections:
  +  +  +  +  +          a b c d e   f g h i j
  -  -  -  -  -         ┌─────────┬─────────┐
    (vertical)          │ 1 ●●●●● │ ●●●●● 1 │
                        │ 2 ●●●●● │ ●●●●● 2 │
                        └─────────┴─────────┘
```
- Κάθε σειρά a-e και f-j συνδέονται οριζόντια
- Οι κάθετες ράγες (+/-) συνδέονται κάθετα

## Οδηγός Θεμάτων

### 1) Εγκατάσταση του Arduino IDE
- Κατεβάστε και εγκαταστήστε το Arduino IDE για το λειτουργικό σας σύστημα από το arduino.cc.
- Συνδέστε τη πλακέτα (π.χ. Arduino Uno/Uno R3/Clone). Για κλώνους ίσως χρειαστούν drivers (π.χ. CH340/CP2102).
- IDE: Tools → Board → επιλέξτε το σωστό board. Tools → Port → επιλέξτε τη θύρα της πλακέτας.
- Δοκιμή ανεβάσματος: Φορτώστε το παρακάτω Blink και πατήστε Upload.

**Αρχείο sketch:** `./blink.ino`

**Συνδεσμολογία:**
```
┌─────────────────┐
│   Arduino Uno   │
│                 │
│  D13 (Built-in) ├──► LED εσωτερικό της πλακέτας
│                 │
│      GND        │
└─────────────────┘
```

- Δεν απαιτείται εξωτερική καλωδίωση.
- Το `LED_BUILTIN` (pin 13) έχει ενσωματωμένο LED και αντίσταση.
- Απλά ανεβάστε τον κώδικα και παρατηρήστε το LED να αναβοσβήνει.

### 2) Κατανόηση ψηφιακού I/O
- `pinMode(pin, INPUT|OUTPUT|INPUT_PULLUP)`: Ρύθμιση τρόπου λειτουργίας ακίδας.
- `digitalWrite(pin, HIGH|LOW)`: Εγγραφή λογικού επιπέδου σε ακίδα OUTPUT.
- `digitalRead(pin)`: Ανάγνωση λογικού επιπέδου από ακίδα INPUT.
- Κουμπιά: Προτιμήστε `INPUT_PULLUP` για εσωτερική αντίσταση. Η λογική αντιστρέφεται (πατημένο = LOW).

**Αρχείο sketch:** `./button_led.ino`

**Συνδεσμολογία:**
```
           ┌─────────────────┐
           │   Arduino Uno   │
           │                 │
  ┌────────┤ D2 (INPUT_PULLUP│
  │        │                 │
  │        │  D13 (Built-in) ├──► LED εσωτερικό
  │        │                 │
  │    ┌───┤ GND             │
  │    │   └─────────────────┘
  │    │
  └─┬──┘      Κουμπί (Push Button)
    └─────     Πατήστε για σύνδεση D2 με GND
```

**Υλικά:**
- 1× Push Button (κουμπί)
- 2× Jumper wires (καλώδια)

**Βήματα:**
1. Συνδέστε ένα άκρο του κουμπιού στο D2.
2. Συνδέστε το άλλο άκρο του κουμπιού στο GND.
3. Χρησιμοποιούμε `INPUT_PULLUP` (εσωτερική αντίσταση pull-up), άρα δεν χρειάζεται εξωτερική αντίσταση.
4. Όταν πατάτε το κουμπί, το D2 συνδέεται με GND (= LOW) και ανάβει το LED.

### 3) Βασικές έννοιες προγραμματισμού
- Μεταβλητές/τύποι: `int`, `float`, `bool`, `const`, `unsigned long` (χρήσιμο για χρόνους).
- Ροή ελέγχου: `if/else`, `for`, `while`, `switch`.
- Συναρτήσεις: ορισμός/κλήση, πεδία ορατότητας μεταβλητών (scope).
- Χρονισμός: Προτιμήστε μη-αποκλειστικό χρονισμό με `millis()` αντί για πολλαπλά `delay()`.

**Αρχείο sketch:** `./millis_blink.ino`

**Συνδεσμολογία:**
```
┌─────────────────┐
│   Arduino Uno   │
│                 │
│  D13 (Built-in) ├──► LED εσωτερικό (χρήση millis)
│                 │
│      GND        │
└─────────────────┘
```

- Ίδια συνδεσμολογία με το βασικό Blink (εσωτερικό LED).
- Η διαφορά είναι στον κώδικα: χρήση `millis()` αντί για `delay()`.
- Επιτρέπει παράλληλη εκτέλεση άλλου κώδικα χωρίς μπλοκάρισμα.

### 4) Παραλλαγές αναβοσβήματος LED
- Βασικό Blink: `digitalWrite` + `delay` (βλέπε παραπάνω).
- Fade (PWM): `analogWrite(pinPWM, value)` με τιμές 0–255. Προσοχή: μόνο σε ακίδες με σήμανση `~`.
- Πολλαπλά LEDs: μη-αποκλειστικός χρονισμός με `millis()` για ανεξάρτητα μοτίβα.

**Αρχείο sketch:** `./pwm_fade.ino`

**Συνδεσμολογία:**
```
           ┌─────────────────┐
           │   Arduino Uno   │
           │                 │
           │      D9 (~)     ├────┐
           │                 │    │
           │      GND        ├──┐ │
           └─────────────────┘  │ │
                                │ │
                              ┌─┴─┴─┐
                    220Ω      │ R   │
                    ┌─────────┤     │
                    │         └─────┘
                    │
                 ───▼───  LED (κόκκινο/πράσινο)
                 ───────  (Anode +)
                    │
                    │     (Cathode -)
                    └──────────► GND
```

**Υλικά:**
- 1× LED (οποιοδήποτε χρώμα)
- 1× Αντίσταση 220Ω (κόκκινο-κόκκινο-καφέ)
- 2× Jumper wires

**Βήματα:**
1. Συνδέστε το **μακρύ ποδαράκι** (anode +) του LED με την αντίσταση 220Ω.
2. Συνδέστε την αντίσταση στο **D9** (PWM pin με ~ σήμανση).
3. Συνδέστε το **κοντό ποδαράκι** (cathode -) του LED στο **GND**.
4. Το LED θα fade σταδιακά από σβηστό σε φωτεινό και πίσω.

### 5) Βασικά σειριακής επικοινωνίας
- Εκκίνηση: `Serial.begin(9600);` και Serial Monitor με ίδιο baud rate.
- Εκτύπωση: `Serial.print/println` για αποσφαλμάτωση και logs.
- Ανάγνωση: `Serial.available()`, `Serial.read()`/`readString()`, `parseInt()`.

**Αρχείο sketch:** `./serial_led.ino`

**Συνδεσμολογία:**
```
┌─────────────────┐         ╔═══════════════╗
│   Arduino Uno   │  USB    ║   Computer    ║
│                 ├─────────║               ║
│  D13 (Built-in) ├──► LED  ║  Serial       ║
│                 │         ║  Monitor      ║
│      GND        │         ║  (9600 baud)  ║
└─────────────────┘         ╚═══════════════╝
```

**Υλικά:**
- Μόνο το καλώδιο USB (για σειριακή επικοινωνία και τροφοδοσία)

**Βήματα:**
1. Συνδέστε το Arduino στον υπολογιστή με USB.
2. Ανεβάστε το sketch.
3. Ανοίξτε το Serial Monitor (Tools → Serial Monitor ή Ctrl+Shift+M).
4. Ρυθμίστε baud rate σε **9600**.
5. Στείλτε `1` για να ανάψει το LED, `0` για να σβήσει.
6. Θα δείτε μηνύματα "LED ON" / "LED OFF" στο Serial Monitor.

### 6) Τεχνικές αποσφαλμάτωσης (Troubleshooting)
- IDE/Σύνδεση: Ελέγξτε Board/Port, καλώδιο USB δεδομένων, drivers (ιδίως για κλώνους), τροφοδοσία.
- Upload errors: Πατήστε Reset πριν το upload (σε ορισμένους κλώνους), κλείστε άλλες εφαρμογές που χρησιμοποιούν τη θύρα.
- Wiring: Κοινή γείωση, σωστή πολικότητα LED, αντίσταση ~220Ω σε σειρά με LED.
- Serial Debug: Προσθέστε στοχευμένα `Serial.println()` για τιμές/καταστάσεις. Ξεκινήστε από ελάχιστο παράδειγμα και προσθέστε σταδιακά.
- Κουμπιά: Χρησιμοποιήστε `INPUT_PULLUP` και αντιστρέψτε τη λογική. Για «τρεμάμενες» αναγνώσεις, εφαρμόστε debounce (λογισμικό ή RC).

> Σημείωση για αναλογικά: `analogRead(A0)` επιστρέφει 0–1023. Το `analogWrite()` είναι PWM (όχι DAC) και δέχεται 0–255 σε pins με ~.
