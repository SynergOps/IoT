# 01_Basics

## Εισαγωγή στο Arduino

Αυτός ο φάκελος περιέχει βασικά έργα για να ξεκινήσετε με έννοιες Arduino και IoT.

### Θέματα που Καλύπτονται
- Εγκατάσταση του Arduino IDE
- Κατανόηση ψηφιακού I/O
- Βασικές έννοιες προγραμματισμού
- Παραλλαγές αναβοσβήματος LED
- Βασικά σειριακής επικοινωνίας
- Τεχνικές αποσφαλμάτωσης

---

## Οδηγός Θεμάτων

### 1) Εγκατάσταση του Arduino IDE
- Κατεβάστε και εγκαταστήστε το Arduino IDE για το λειτουργικό σας σύστημα από το arduino.cc.
- Συνδέστε τη πλακέτα (π.χ. Arduino Uno/Uno R3/Clone). Για κλώνους ίσως χρειαστούν drivers (π.χ. CH340/CP2102).
- IDE: Tools → Board → επιλέξτε το σωστό board. Tools → Port → επιλέξτε τη θύρα της πλακέτας.
- Δοκιμή ανεβάσματος: Φορτώστε το παρακάτω Blink και πατήστε Upload.

Αρχείο sketch: `./blink.ino`

### 2) Κατανόηση ψηφιακού I/O
- `pinMode(pin, INPUT|OUTPUT|INPUT_PULLUP)`: Ρύθμιση τρόπου λειτουργίας ακίδας.
- `digitalWrite(pin, HIGH|LOW)`: Εγγραφή λογικού επιπέδου σε ακίδα OUTPUT.
- `digitalRead(pin)`: Ανάγνωση λογικού επιπέδου από ακίδα INPUT.
- Κουμπιά: Προτιμήστε `INPUT_PULLUP` για εσωτερική αντίσταση. Η λογική αντιστρέφεται (πατημένο = LOW).

Αρχείο sketch: `./button_led.ino`

### 3) Βασικές έννοιες προγραμματισμού
- Μεταβλητές/τύποι: `int`, `float`, `bool`, `const`, `unsigned long` (χρήσιμο για χρόνους).
- Ροή ελέγχου: `if/else`, `for`, `while`, `switch`.
- Συναρτήσεις: ορισμός/κλήση, πεδία ορατότητας μεταβλητών (scope).
- Χρονισμός: Προτιμήστε μη-αποκλειστικό χρονισμό με `millis()` αντί για πολλαπλά `delay()`.

Αρχείο sketch: `./millis_blink.ino`

### 4) Παραλλαγές αναβοσβήματος LED
- Βασικό Blink: `digitalWrite` + `delay` (βλέπε παραπάνω).
- Fade (PWM): `analogWrite(pinPWM, value)` με τιμές 0–255. Προσοχή: μόνο σε ακίδες με σήμανση `~`.
- Πολλαπλά LEDs: μη-αποκλειστικός χρονισμός με `millis()` για ανεξάρτητα μοτίβα.

Αρχείο sketch: `./pwm_fade.ino`

### 5) Βασικά σειριακής επικοινωνίας
- Εκκίνηση: `Serial.begin(9600);` και Serial Monitor με ίδιο baud rate.
- Εκτύπωση: `Serial.print/println` για αποσφαλμάτωση και logs.
- Ανάγνωση: `Serial.available()`, `Serial.read()`/`readString()`, `parseInt()`.

Αρχείο sketch: `./serial_led.ino`

### 6) Τεχνικές αποσφαλμάτωσης (Troubleshooting)
- IDE/Σύνδεση: Ελέγξτε Board/Port, καλώδιο USB δεδομένων, drivers (ιδίως για κλώνους), τροφοδοσία.
- Upload errors: Πατήστε Reset πριν το upload (σε ορισμένους κλώνους), κλείστε άλλες εφαρμογές που χρησιμοποιούν τη θύρα.
- Wiring: Κοινή γείωση, σωστή πολικότητα LED, αντίσταση ~220Ω σε σειρά με LED.
- Serial Debug: Προσθέστε στοχευμένα `Serial.println()` για τιμές/καταστάσεις. Ξεκινήστε από ελάχιστο παράδειγμα και προσθέστε σταδιακά.
- Κουμπιά: Χρησιμοποιήστε `INPUT_PULLUP` και αντιστρέψτε τη λογική. Για «τρεμάμενες» αναγνώσεις, εφαρμόστε debounce (λογισμικό ή RC).

> Σημείωση για αναλογικά: `analogRead(A0)` επιστρέφει 0–1023. Το `analogWrite()` είναι PWM (όχι DAC) και δέχεται 0–255 σε pins με ~.
