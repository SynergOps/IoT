# 01_Basics

## Εισαγωγή στο Arduino

Αυτός ο φάκελος περιέχει βασικά έργα για να ξεκινήσετε με έννοιες Arduino και IoT.

### Θέματα που Καλύπτονται
- Εγκατάσταση του Arduino IDE
- Κατανόηση ψηφιακού I/O
- Βασικές έννοιες προγραμματισμού
- Παραλλαγές αναβοσβήματος LED
- Βασικά σειριακής επικοινωνίας
- Τεχνικές αποσφαλμάτωσης

### Δείγματα Έργων
- Hello World (Αναβόσβημα LED)
- Επικοινωνία Serial Monitor
- Ψηφιακή Ανάγνωση/Εγγραφή
- Αναλογική Ανάγνωση/Εγγραφή
- Χρονισμός και Καθυστερήσεις

### Μαθησιακοί Στόχοι
- Κατανόηση της εγκατάστασης υλικού και λογισμικού Arduino
- Εκμάθηση βασικών δομών προγραμματισμού (μεταβλητές, βρόχοι, συνθήκες)
- Κυριαρχία στις λειτουργίες ψηφιακού και αναλογικού I/O
- Αποσφαλμάτωση προγραμμάτων χρησιμοποιώντας το Serial Monitor
