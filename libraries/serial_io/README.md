# serial_io Library

Η βιβλιοθήκη `serial_io` παρέχει ένα σύνολο βοηθητικών συναρτήσεων για την απλοποίηση της επικοινωνίας μέσω της σειριακής θύρας (Serial Port) στο Arduino.

## Άδεια Χρήσης

Η βιβλιοθήκη διατίθεται ως **Public Domain**. Αυτό σημαίνει ότι είναι ελεύθερη για χρήση, τροποποίηση και διανομή χωρίς κανέναν περιορισμό. Δείτε το αρχείο `LICENSE` για περισσότερες λεπτομέρειες.

## Εγκατάσταση

1. Κατεβάστε το αρχείο `serial_io.zip` της βιβλιοθήκης.
2. Ανοίξτε το **Arduino IDE**.
3. Πηγαίνετε στο μενού **Sketch** -> **Include Library** -> **Add .ZIP Library...**
4. Επιλέξτε το αρχείο `serial_io.zip` που κατεβάσατε.
5. Η βιβλιοθήκη είναι πλέον εγκατεστημένη και έτοιμη για χρήση!

## Δομή Αρχείων

* **`serial_io.h`**: Το κύριο αρχείο της βιβλιοθήκης που περιέχει όλο τον πηγαίο κώδικα.
* **`keywords.txt`**: Αρχείο για τον χρωματισμό συντάξεως (syntax highlighting) στο Arduino IDE.
* **`library.properties`**: Μεταδεδομένα της βιβλιοθήκης (όνομα, έκδοση, συγγραφείς) για τον Library Manager.
* **`LICENSE`**: Το κείμενο της άδειας χρήσης (Public Domain / Unlicense).
* **`examples/`**: Φάκελος με παραδείγματα χρήσης (π.χ. `Serial_IO_Demo`).
* **`serial_io.zip`**: Το αρχείο λήψης και εγκατάστασης της βιβλιοθήκης στο Arduino IDE.

## Λειτουργίες

### 1. Συναρτήσεις Εξόδου (Εκτύπωση)
Οι συναρτήσεις αυτές επιτρέπουν την εύκολη εκτύπωση τιμών στη σειριακή οθόνη.
* **`print(...)`**: Εκτυπώνει από 1 έως 5 ορίσματα (μεταβλητές ή κείμενο) στην ίδια γραμμή, χωρισμένα με κενά.
    * Διαχειρίζεται αυτόματα δεκαδικούς αριθμούς (`float`, `double`) τυπώνοντάς τους με 2 δεκαδικά ψηφία (προεπιλογή).
* **`println(...)`**: Λειτουργεί όπως η `print`, αλλά αλλάζει γραμμή στο τέλος.
* **`println()`**: Εκτυπώνει απλώς μια κενή γραμμή.

### 2. Συναρτήσεις Εισόδου (Ανάγνωση)
Οι συναρτήσεις αυτές σταματούν την εκτέλεση του προγράμματος και περιμένουν τον χρήστη να πληκτρολογήσει κάτι στη σειριακή οθόνη.
* **`inputString(prompt, echo)`**: Εμφανίζει ένα μήνυμα προτροπής (`prompt`), περιμένει να δώσει ο χρήστης κείμενο και το επιστρέφει ως `String`.
* **`inputCharacter(prompt)`**: Ζητάει είσοδο και επιστρέφει τον πρώτο χαρακτήρα που πληκτρολογήθηκε.
* **`inputInteger(prompt)`**: Ζητάει είσοδο και μετατρέπει το κείμενο σε ακέραιο αριθμό (`long`).
* **`inputFloat(prompt)`**: Ζητάει είσοδο και μετατρέπει το κείμενο σε δεκαδικό αριθμό (`float`).

### 3. Συναρτήσεις Ελέγχου
* **`stop()`**: Τερματίζει τη ροή του προγράμματος. Εκτυπώνει ένα μήνυμα τέλους (στα Ελληνικά) και μπαίνει σε έναν ατέρμονα βρόχο (`while(true)`), σταματώντας ουσιαστικά το Arduino μέχρι να γίνει Reset.

## Παράδειγμα Χρήσης

```cpp
// Προαιρετικά: Ορισμός δεκαδικών ψηφίων (προεπιλογή = 2)
#define DECIMAL_PLACES 4
#include "serial_io.h"

void setup() {
  Serial.begin(9600);
  
  // 1. Παραδείγματα Εξόδου (print/println)
  println("--- Επίδειξη Εξόδου ---");
  print("Απλή εκτύπωση χωρίς αλλαγή γραμμής. ");
  println("Τώρα αλλάζουμε γραμμή.");
  
  // Εκτύπωση πολλαπλών ορισμάτων (έως 5)
  println("Μέτρηση:", 1, "Τιμή:", 12.3456789, "Μονάδες"); // Θα δείξει 4 δεκαδικά λόγω του define
  println(); // Κενή γραμμή
  delay(2000);
  // 2. Παραδείγματα Εισόδου
  println("--- Επίδειξη Εισόδου ---");
  
  // inputString (με και χωρίς echo)
  String name = inputString("Πώς σε λένε; "); // echo = true (default)
  String secret = inputString("Δώσε έναν κωδικό (κρυφό): ", false); // echo = false
  println("\nΟ κωδικός καταχωρήθηκε."); // Χρειάζεται \n γιατί το echo=false δεν αλλάζει γραμμή

  // inputCharacter
  char answer = inputCharacter("Σου αρέσει ο προγραμματισμός; (y/n): ");
  
  // inputInteger & inputFloat
  int age = inputInteger("Πόσο χρονών είσαι; ");
  float height = inputFloat("Τι ύψος έχεις (σε μέτρα); ");

  // 3. Εμφάνιση αποτελεσμάτων
  println("\n--- Αποτελέσματα ---");
  println("Όνομα:", name);
  println("Απάντηση:", answer);
  println("Ηλικία:", age);
  println("Ύψος:", height);

  // 4. Τερματισμός
  stop();
}

void loop() {
  // Ο κώδικας εδώ δεν θα εκτελεστεί ποτέ λόγω της stop() στο setup
}
```
